Technical Overview:
A visually rich, fully responsive landing page designed for a niche piano restoration and retail business. The project demonstrates a strong command of modern CSS layouts and user-centric UI design.

Custom Styling: Utilizes CSS Variables (:root) for consistent branding and Font Awesome for professional iconography.
